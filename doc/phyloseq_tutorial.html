Importing Nephele Results into Phyloseq

These are some general instructions for how to import the outputs from Nephele into phyloseq.

Software requirements

Please refer to the websites linked here for installation and usage instructions:

  • R
  • phyloseq - installing phyloseq will also install the other packages used in this tutorial.
  • RStudio - not required, but helpful if you are new to R.

Tutorial

  1. Download and unzip the PipelineResults folder from your Nephele job. This folder contains all of the data and files used in processing your job.
  2. In the files pane of RStudio navigate to your unzipped outputs folder and under the "More" dropdown, choose Set As Working Directory.

  3. In the Console, enter the following:

    library(phyloseq)
  4. To import the data as a phyloseq object, use phyloseq's import_biom command.

  • DADA2

    mydata <- import_biom(BIOMfilename = "taxa_species.biom")
mydata
    ## phyloseq-class experiment-level object
## otu_table()   OTU Table:         [ 516 taxa and 11 samples ]
## tax_table()   Taxonomy Table:    [ 516 taxa by 7 taxonomic ranks ]

  • QIIME
    You may see a warning message when importing the biom file from QIIME.

    mydata <- import_biom(BIOMfilename = "otus/otu_table_mc2_w_tax_no_pynast_failures.biom")
    ## Warning in strsplit(conditionMessage(e), "\n"): input string 1 is invalid
## in this locale
    mydata
    ## phyloseq-class experiment-level object
## otu_table()   OTU Table:         [ 6567 taxa and 11 samples ]
## tax_table()   Taxonomy Table:    [ 6567 taxa by 7 taxonomic ranks ]

  1. Import the sample metadata with import_qiime_sample_data and merge it with the phyloseq object. Subsititute the name of your mapping file for map_file.txt inside the quotes.

    mapfile <- import_qiime_sample_data("map_file.txt")
mydata <- merge_phyloseq(mydata, mapfile)
mydata
    ## phyloseq-class experiment-level object
## otu_table()   OTU Table:         [ 6567 taxa and 10 samples ]
## sample_data() Sample Data:       [ 10 samples by 7 sample variables ]
## tax_table()   Taxonomy Table:    [ 6567 taxa by 7 taxonomic ranks ]
  2. (Optional) To add a tree to the phyloseq object, use the ape package to read in the tree file, and then add it to mydata using phy_tree.

  • QIIME

    library(ape)
tree <- read.tree("otus/rep_set.tre")
